beanshell      starts a beanshell.  Type 'importCommands("bsh");' to get
               access to the beanshell commands in subdirectory bsh.
ozed2unicode   an Object Z latex to unicode converter.
ozed2zml       an Object Z latex to zml converter.
unicode2latex  a Standard Z unicode to latex converter.
zed2unicode    a Standard Z latex to unicode converter.
zed2zml        a Standard Z latex to zml converter.

To get the command line tools working, call 'ant' or 'ant install-bin'
in the CZT_HOME directory (that is the parent of the directory where
this README file is in).
