Implementation classes of the AST for Z Pattern.

This package provides implementations of the interfaces in {@link net.sourceforge.czt.zpatt.ast}. As an application programmer, you should ONLY use the object factory provided in this package explicitly. Use the interfaces in {@link net.sourceforge.czt.zpatt.ast} and the {@link net.sourceforge.czt.zpatt.impl.ZpattFactoryImpl} when using the AST for Z Pattern.
